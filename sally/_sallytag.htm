기본태그 응용편

  • 평면액자 - 가는 테
  • 입체액자 - 굵은 테
  • 입체액자 - 글주위 다른색(glow)
  • 영상시 - 글이동(마끼)+배경
  • 영상시 - 글이동(마끼)+배경흐림
  • 영상시 - 글이동(마끼)+배경흐림+배경음악
  • 테이블(표) - 사진,글 이어서 넣기
  • 플래시 넣기 - 주소입력
  • 동영상 넣기 - 주소입력
  • 음악 넣기 - 주소입력,버튼감춤
  • 음악 넣기 - 주소입력,버튼일부보임
  • 음악 넣기 - 주소입력,버튼전체보임


많이 쓰이는 HTML태그

  • 태그 체험-클릭!(소스넣기버튼도 클릭)
  • A 앵커 - 클릭하여 웹페이지 열기
  • B 볼드 - 글씨를 굵게
  • CENTER 와 ALIGN - 글이나 이미지 정렬
  • BR 과   - 줄바꾸기와 공백
  • INPUT과 SELECT - 폼태그 (1)
  • TEXTAREA - 폼태그 (2)
  • TABLE - 테이블(표) 셀(칸) 나누기
  • TABLE - 테이블(표) 꾸미기
  • IMG - 이미지 넣기
  • FONT - 글씨체,크기,색상,글자배경
